# Changelog

## v1.0.0 - Initial Release

Premiere sortie publique de **WorldScalingMobs**.

Cette version introduit le systeme principal du datapack : une difficulte progressive basee sur le nombre de jours passes dans le monde. Les mobs hostiles peuvent maintenant apparaitre sous differentes variantes, allant de simples versions renforcees jusqu'aux elites de Tier 5.

## Ajouts

- Systeme de progression en 5 periodes :
  - Periode I - Eveil
  - Periode II - Menace
  - Periode III - Siege
  - Periode IV - Cataclysme
  - Periode V - Apocalypse
- Apparition aleatoire de mobs classes par tiers selon la periode actuelle.
- Stats personnalisees pour les mobs renforces : vie, vitesse, force, resistance, regeneration ou resistance au feu.
- Equipements personnalises selon le type de mob et son tier.
- Mobs de Tier 5 avec nom special, effet lumineux et bonus avances.
- Tables de loot personnalisees pour chaque mob et chaque tier.
- Bossbar affichant la periode actuelle du monde.
- Systemes internes de detection, roll aleatoire et application automatique des tiers.

## Notes

- Le datapack fonctionne cote serveur et en solo.
- Aucun mod client n'est requis.
- La difficulte augmente automatiquement avec les jours du monde.
- Les recompenses deviennent plus interessantes lorsque les mobs deviennent plus dangereux.
